## Simple fullstack app that plays with express in the BE and Vite in the FE

### commands ran for BE scaffolding:

mkdir full-stack-app folder
inside folder created a README file and other folder "server"
inside server ran: npm init -y

npm i express@^4.18.2 cors@^2.8.5

npm i @types/express@^4.17.21 @types/cors@^2.8.19 typescript@^5.3.3 nodemon@^3.1.10 ts-node@^10.9.2 -D

npx tsc --init
git init
touch .gitignore

### commands for FE scaffolding

mkdir front && cd front
npm create vite@latest .
choose react and typescript
